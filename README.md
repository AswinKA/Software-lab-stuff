# Crop Disease Detection and Prevention System
The Crop Disease Detection and Prevention System is a web-based platform that serves as a comprehensive resource for crop disease information. Users can browse crops and their associated diseases, learn about symptoms, causes, prevention methods, and recommended treatments. The platform also includes community features such as comments and direct messaging, allowing users to share knowledge, ask questions, and connect with others. Administrators manage crop and disease records, moderate user interactions, and oversee the platform to ensure accurate and reliable information.
## Features

### User
- User registration and login
- Browse a catalog of crops and their associated diseases
- View detailed disease information, including symptoms, causes, and affected crops
- Access prevention and treatment recommendations
- Search for crops and diseases
- Comment on disease pages and discussions
- Send and receive direct messages (DM)
- View user profiles and activity

### Admin
- Admin dashboard
- Manage users
- Ban and unban users
- Add, edit, and delete crop information
- Add, edit, and delete disease records
- Moderate comments and discussions
- Manage user-reported content
- View platform statistics
